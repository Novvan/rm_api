# rm_api

Created to show the use of React-Typescript with the Rick and morty API and MUI (Material Ui)

To start poject in dev environment:
```
 yarn dev 
```
or, if you want to expose it to your network
```
 yarn dev --host 
```
to build it use 
```
 yarn build
```
and lastly, to serve 
```
 yarn serve
```
